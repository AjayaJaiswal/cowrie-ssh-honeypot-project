# SSH Honeypot Deployment using Cowrie

This project demonstrates the deployment of an SSH honeypot using the Cowrie framework to capture attacker interactions and analyze malicious behavior.

## Objective
Simulate a vulnerable SSH server and record attacker activities such as login attempts, executed commands, and session behavior.

## Environment
- Kali Linux Virtual Machine
- Cowrie SSH Honeypot
- Python
- SSH

## Steps Performed
1. Installed and configured Cowrie honeypot
2. Deployed honeypot on a Linux VM
3. Simulated attacker SSH login attempts
4. Captured attacker commands and session activity
5. Analyzed logs generated by the honeypot

## Key Findings
The honeypot logs captured:
- Attacker IP addresses
- Username and password attempts
- Commands executed during the session
- SSH session activity

This project demonstrates practical experience with honeypot deployment and attacker behavior monitoring.
